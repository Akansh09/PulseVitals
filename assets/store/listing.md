## Long description

```
PulseVitals shows you how fast your page really is — without opening DevTools.

One click on the toolbar icon pops up a 380×520 report with:

• Core Web Vitals verdict (Good / Needs improvement / Poor) with a green / amber / red dot on the toolbar icon that updates as pages load. Based on Google's published LCP, CLS, and INP thresholds — the same ones Search uses for ranking.

• Plain-English fixes. Eight rule-based insights translate "your LCP is 4.8 s" into "your hero image is 2.1 MB — compress to WebP and serve a responsive srcset." Each suggestion includes an estimated savings in seconds.

• Third-party origin breakdown. Groups every resource on the page by host and names the big ones (Google, Meta, HubSpot, Hotjar, Intercom, and more). Finally answers "what are my tags costing me?"

• SEO sanity checks. Title, meta description, H1 count, canonical, Open Graph, viewport, and JSON-LD — pass/fail for each, in the same popup.

• Accessibility + CWV combined score. A single number (0-100) combining Core Web Vitals pass and a bundled lightweight axe rule set.

• Batch audit. Paste up to 50 URLs, get a ranked CWV table you can export to CSV.

• Competitor comparison. Enter up to 3 competitor URLs, get a head-to-head chart you can share as a 1200×630 PNG.

• History. The last 10 runs per origin, stored locally. Reopen any snapshot; see a histogram of your LCP / CLS / INP across recent visits.

• Share. One click produces a self-contained HTML report — open it in any browser, email it to a client, archive it for later. No account, no hosted link.

• Privacy-first. Zero telemetry. No data ever leaves your device unless you explicitly Export, Share, or enable desktop budget alerts. Full privacy policy shipped with the extension.

PulseVitals is MIT-licensed, open-source, and doesn't require sign-up. Ideal for:

• Site owners and founders who want to know "is my site fast for Google Search?"
• Consultants running client audits — share a branded HTML report in a click.
• Marketers and ad-ops curious which tags are slowing the site.
• Developers who want a quick Core Web Vitals glance without opening DevTools.
• Teams working on Shopify, WordPress, Webflow, or Next.js — platform-specific tips appear automatically when the extension detects your stack.

Available in English and Spanish.
```
